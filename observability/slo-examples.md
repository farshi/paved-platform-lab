# SLO Examples

Use these simple SLOs in the lab.

## Availability

- SLI: successful requests / total requests
- SLO: 99.9 percent over 30 days

## Latency

- SLI: requests under 300ms / total requests
- SLO: 95 percent of requests under 300ms over 30 days

## Error Rate

- SLI: 5xx responses / total requests
- SLO: below 1 percent over 30 days

## Burn Rate

- Example alert: fast burn if error rate exceeds 2x budget over 5m and 1h windows
